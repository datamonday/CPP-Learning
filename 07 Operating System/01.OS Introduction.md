# 1. 什么是操作系统？

操作系统（Operating System，OS）是计算机硬件和应用之间的一层软件，方便用户使用硬件。OS管理的硬件包括：

- CPU管理
- 内存管理
- 终端管理
- 磁盘管理
- 文件管理
- 网络管理
- 电源管理
- 多核管理

其中，后三个在高级操作系统课程中会涉及。
